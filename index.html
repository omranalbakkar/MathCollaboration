What's inside
Built for the real classroom.

Smart Question Bank
Create open-text, multiple choice, true/false, and numeric questions. Set points, correct answers, and subject tags.
🔴
Live Student Monitor
See every student's progress in real time. Watch answers roll in the moment students submit, from any device.
⭐
Instant Grading
Review each student's answers per question. Add scores and written feedback that students see on their completion screen.

How it works
Up and running in minutes.
01
Teacher configures session
Log in to the teacher dashboard, set the grade, class, date and timing, then add your questions to the bank.
02
Share the student link
Send students the URL to student.html. They enter their name and join instantly — no accounts, no passwords.
03
Students answer live
The teacher's Live Monitor shows each student's progress in real time. Every answer syncs across all devices instantly via Firebase.
04
Review, grade & feedback
Once submitted, teachers review each student's answers, assign scores, and write feedback — which students see immediately on their screen.
